## Interaction Contract

### Date Range Picker

- Default selection: "Last 30 days" on page load
- Quick presets rendered as toggle group: Last 7d | 30d | 90d | This month | This year
- Active preset highlighted with accent background (#0F766E at 15% opacity, teal text)
- "Custom" opens a Popover with dual Calendar for start/end date selection
- Changing date range triggers re-fetch of all chart data simultaneously
- While loading: charts show Skeleton placeholders, KPI values show Skeleton

### Chart Interactivity

- **Hover:** Crosshair cursor. Tooltip appears showing exact value, date, and category.
- **Click data point:** Opens drill-down panel for that date/period. Clicked element gets accent ring highlight.
- **Click legend item (Revenue/Trends):** Toggles series visibility. Disabled series grayed out in legend.
- **Occupancy heatmap:** Click a day cell to drill down to that day's bookings.
- **Revenue bar:** Click a bar segment to drill down to that room type's bookings for the period.
- **Trends line:** Click a point to drill down to that day's bookings.

### Export

- CSV export: Immediate download of current chart data as .csv file. Filename format: `hotelbook-{chart-name}-{start-date}-{end-date}.csv`
- PDF export: Captures full dashboard viewport as PDF. Shows brief "Generating PDF..." toast (Sonner) during capture. Filename: `hotelbook-reports-{start-date}-{end-date}.pdf`

### Loading States

- Initial page load: All 4 KPI cards show Skeleton, all 3 chart containers show Skeleton rectangles matching chart dimensions
- Date range change: Same skeleton treatment while new data loads
- Individual chart error: Show inline error message within that chart container; other charts unaffected

### Empty States

- Per-chart empty: "No data for this period" centered in chart area with muted text
- Full page empty (no data at all): Use EmptyState component with chart-specific messaging

## Accessibility

| Requirement | Implementation |
|-------------|---------------|
| Chart keyboard nav | Nivo provides aria-label on SVG elements; supplement with visually-hidden summary table per chart |
| Screen reader | Each chart container has `role="img"` with `aria-label` describing the chart type and current date range |
| Color contrast | All text meets WCAG AA (4.5:1) against dark backgrounds. Heatmap uses both color intensity and cell text values |
| Focus management | Drill-down panel traps focus when open; returns focus to triggering chart element on close |
| Reduced motion | Respect `prefers-reduced-motion`: disable Nivo chart animations when enabled |
