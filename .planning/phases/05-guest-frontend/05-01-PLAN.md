   ```bash
   npm install react-router @tanstack/react-query zustand axios date-fns
   npm install react-hook-form @hookform/resolvers zod

   ```

    - `getRoomTypeDetail(id: string, checkIn: string, checkOut: string, guests?: number): Promise<RoomTypeDetail>` - GET /api/v1/search/room-types/{id}

    - `createBooking(data: BookingCreate): Promise<BookingResponse>` - POST /api/v1/bookings/

    - Actions: setStep(n), setBookingId(id), setSearchParams(checkIn, checkOut, guests), setSelectedRoom(room), setGuestDetails(details), reset()
