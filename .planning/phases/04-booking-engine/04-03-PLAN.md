    1. Add to `services/booking/app/services/booking.py` (append to existing):
       - `async def list_bookings(db: AsyncSession, user_id: UUID, status_filter: str | None = None, skip: int = 0, limit: int = 20) -> tuple[list[Booking], int]`:
         - Query bookings WHERE user_id = user_id
         - If status_filter provided, add WHERE status = status_filter
         - Order by created_at DESC
         - Return (list of bookings, total count)
         - Perform on-demand expiry check on any PENDING bookings in results
       - `async def cancel_booking(db: AsyncSession, booking_id: UUID, user_id: UUID) -> Booking`:
         - Get booking with ownership check
         - Verify status allows cancellation (pending or confirmed only, per VALID_TRANSITIONS)
         - If status is "confirmed": calculate cancellation fee
           - If today >= check_in - cancellation_policy_days: fee = booking.price_per_night (first night charge)
           - Else: fee = None (free cancellation)
           - Set booking.cancellation_fee = fee
         - Call transition_booking(db, booking, "cancelled", reason="guest_cancelled")
         - Return booking
       - `async def modify_booking(db: AsyncSession, booking_id: UUID, user_id: UUID, data: BookingModifyRequest) -> dict`:
         - Get booking with ownership check
         - Verify status is "confirmed" (cannot modify PENDING, CHECKED_IN, etc.)
         - Determine new values: use data fields if provided, else keep existing values
         - If check_in, check_out, room_type_id, or guest_count changed:
           - Re-check availability using pessimistic locking (same pattern as create_booking, but exclude current booking from blocking count)
           - Re-fetch pricing from Room service
           - Calculate price difference
         - Update booking fields (dates, room_type, guest_count, guest details)
         - Update price snapshot if pricing changed
         - Publish "booking.modified" event (if dates/room changed) -- note: Room service consumer will process this as an update to existing projection
         - await db.commit(), await db.refresh(booking)
         - Return dict with: booking, old_total, new_total, price_difference

         - Inside loop: create async session from async_session_factory
         - Query all bookings WHERE status="pending" AND expires_at < now(UTC)
         - For each expired booking: call transition_booking(db, booking, "cancelled", reason="expired")
         - Log count of expired bookings
