    1. `services/chat/app/llm/base.py`: Abstract LLM provider interface:
       ```python
       from abc import ABC, abstractmethod
       from typing import AsyncIterator
       from dataclasses import dataclass

       @dataclass
       class StreamChunk:
           type: str  # "text_delta" | "tool_use_start" | "tool_use_input" | "tool_use_end" | "done" | "error"
           text: str | None = None
           tool_name: str | None = None
           tool_id: str | None = None
           tool_input: dict | None = None

       @dataclass
       class TokenUsage:
           input_tokens: int
           output_tokens: int

       class LLMProvider(ABC):
           @abstractmethod
           async def stream_message(
               self,
               messages: list[dict],
               system: str,
               tools: list[dict],
               max_tokens: int = 4096,
           ) -> AsyncIterator[StreamChunk]: ...

           @abstractmethod
           async def get_usage(self) -> TokenUsage: ...
       ```

    2. `services/chat/app/llm/anthropic_provider.py`: Implement `AnthropicProvider(LLMProvider)`:
       - Constructor: `__init__(self, api_key: str, model: str)` -- create `anthropic.AsyncAnthropic(api_key=api_key)`, store model name
       - `stream_message`: Use `self.client.messages.stream(model=..., max_tokens=..., system=..., messages=..., tools=...)`. Iterate async events:
         - `event.type == "text"` -> yield `StreamChunk(type="text_delta", text=event.text)`
         - `event.type == "content_block_start"` with `hasattr(event.content_block, "name")` -> yield `StreamChunk(type="tool_use_start", tool_name=event.content_block.name, tool_id=event.content_block.id)`
         - `event.type == "input_json"` -> yield `StreamChunk(type="tool_use_input", text=event.partial_json)`
         - `event.type == "content_block_stop"` with tool input -> yield `StreamChunk(type="tool_use_end", tool_id=event.content_block.id, tool_input=event.content_block.input)`
       - After stream: `final = await stream.get_final_message()`, store `TokenUsage(input_tokens=final.usage.input_tokens, output_tokens=final.usage.output_tokens)`
       - Yield `StreamChunk(type="done")` at end
       - `get_usage`: Return stored `_usage`
       - Convert tools from provider-agnostic format: each tool dict has `name`, `description`, `input_schema` -- Anthropic takes this natively

    3. `services/chat/app/llm/openai_provider.py`: Implement `OpenAIProvider(LLMProvider)`:
       - Constructor: `__init__(self, api_key: str, model: str)` -- create `openai.AsyncOpenAI(api_key=api_key)`, store model
       - `stream_message`: Use `self.client.chat.completions.create(model=..., max_tokens=..., messages=[{"role":"system","content":system}]+messages, tools=[...], stream=True)`. Convert tool format: each tool -> `{"type":"function","function":{"name":name,"description":desc,"parameters":input_schema}}`
       - Parse streaming chunks: handle `delta.content` for text, `delta.tool_calls` for tool use, `finish_reason` for completion
       - Accumulate tool call arguments from deltas (OpenAI streams tool args incrementally)
       - `get_usage`: Return stored `_usage` from stream final chunk
       - Handle `stream_options={"include_usage": True}` to get token counts
